Part I — Multivariate Descriptive Statistics
01 What is Multivariate Data? Notation & Setup New
📓 From Notes (Note_13_01_26 & Recapitulation)
Definition: By multivariate data we mean data collected on a number of variables from each of a given set of individuals. Let there be 'p' such variables x₁, x₂, …, xₚ and we have n individuals under consideration.

Notation: We denote by xᵢₐ the value of xᵢ for the α-th individual:
i = 1, 2, …, p (variable index) α = 1, 2, …, n (individual index)
Real Dataset Example — 10 Largest U.S. Industrial Corporations (Fortune 500, 1990):
CompanySales x₁ ($M)Profit x₂ ($M)Assets x₃ ($M)
General Motors126,9744,224173,297
Ford96,9333,835160,893
Exxon86,6563,51083,219
IBM63,4383,75877,734
General Electric55,2643,939128,344
Mobil50,9761,80939,080
Philip Morris39,0692,94638,528
Chrysler36,15635951,038

Texaco32,4162,41325,636
Discussion: This is 3-variable (p=3) multivariate data with n=10 observations.
💡 Simple Explanation
In univariate statistics, you study one variable at a time (e.g., just sales). In bivariate, you study two together (sales vs. profit). In multivariate, you study all p variables simultaneously.

The subscript notation xᵢₐ is just a bookkeeping system:
  • First subscript i = which variable (Sales? Profit? Assets?)
  • Second subscript α = which company/individual
So x₁₃ = Sales of Exxon = 86,656 million.
02 Types of Multivariate Descriptive Problems New
📓 From Notes
In the multivariate setup, there are two main types of problems:

Type 1 — Multiple Regression: There is one variable of principal interest, say x₁. We want to judge how x₁ is related to x₂, x₃, …, xₚ. We form an equation expressing x₁ as a function of x₂, x₃, …, xₚ. This equation may then be used as a predicting formula for x₁ in a situation where its value is unknown but the values of the other variables x₂, x₃, …, xₚ are known.

Type 2 — Multiple Correlation: x₁ may be the variable of principal interest and we may like to judge the extent to which x₁ may be said to depend on x₂, x₃, …, xₚ. This is a measure of association.

Type 3 — Partial Correlation: There will be two variables of principal interest, say x₁ and x₂. We want to judge the extent to which x₁ and x₂ are related when the effects of x₃, x₄, …, xₚ are eliminated from each of them. In other words, we judge to what extent x₁ and x₂ may be considered to be related independent of the influence of x₃, x₄, …, xₚ on them.

Note: In every situation we shall consider only the linear relationship.
💡 Simple Explanation
Think of it with the Fortune 500 data (Sales=x₁, Profit=x₂, Assets=x₃):

  • Regression: Predict Sales (x₁) from Profit and Assets. Build a formula.
  • Multiple Correlation: How well do Profit and Assets together predict Sales? Answer: one number R between 0 and 1.
  • Partial Correlation: What is the true relationship between Sales and Profit, after removing the confounding effect of Assets from both?
Key insight: We always deal with linear relationships in this course. Non-linear relationships can often be linearized using log/square-root transformations.
03 Multiple Regression Equation — Full Theory
📓 From Notes (Recapitulation + Handwritten)
Sample Statistics Notation:
x̄ᵢ = (1/n) Σα xᵢα = Mean of xᵢ, i = 1,2,…,p sᵢ² = (1/n) Σα (xᵢα − x̄ᵢ)² = Variance of xᵢ, i = 1,2,…,p sᵢⱼ = (1/n) Σα (xᵢα−x̄ᵢ)(xⱼα−x̄ⱼ) = Cov(xᵢ, xⱼ) rᵢⱼ = sᵢⱼ/(sᵢ sⱼ) = Corr. coeff. between xᵢ and xⱼ rii = 1 for all i
Correlation Matrix:
⎡ 1 r₁₂ r₁₃ … r₁ₚ ⎤ R = ⎢ r₂₁ 1 r₂₃ … r₂ₚ ⎥ ⎣ rₚ₁ rₚ₂ rₚ₃ … 1 ⎦ Rᵢⱼ = Cofactor of the (i,j)th element rᵢⱼ in R
Multiple Regression Equation of x₁ on x₂, x₃, …, xₚ:
X₁.₂₃…ₚ = b₀ + b₂x₂ + b₃x₃ + … + bₚxₚ where: bⱼ = − (R₁ⱼ/R₁₁) · (s₁/sⱼ), j = 2, 3, …, p b₀ = x̄₁ − b₂x̄₂ − b₃x̄₃ − … − bₚx̄ₚ Full form: X₁.₂₃…ₚ = x̄₁ − (R₁₂/R₁₁)(s₁/s₂)(x₂−x̄₂) − (R₁₃/R₁₁)(s₁/s₃)(x₃−x̄₃) − … − (R₁ₚ/R₁₁)(s₁/sₚ)(xₚ−x̄ₚ)
Note on Partial Regression Coefficient: The coefficient bⱼ, or more precisely b₁ⱼ.₂₃…ⱼ₋₁,ⱼ₊₁…ₚ, is called the partial regression coefficient of x₁ on xⱼ, given x₂, x₃, …, xⱼ₋₁, xⱼ₊₁,…, xₚ. It represents the amount by which the predicted value of x₁ increases for one-unit increase in the value of xⱼ, with all other variables remaining unaltered.
💡 Simple Explanation
The regression equation is a linear formula: plug in the values of x₂, x₃, … xₚ and get a prediction of x₁.

How to compute bⱼ step by step:
1
Write down the correlation matrix R (all pairwise correlations)
2
Compute the cofactor R₁ⱼ (the signed minor of element r₁ⱼ in R) for each j=2,…,p
3
Compute the cofactor R₁₁ (the signed minor of the (1,1) element)
4
Apply: bⱼ = −(R₁ⱼ/R₁₁)·(s₁/sⱼ). Note the minus sign!
5
Compute b₀ = x̄₁ − b₂x̄₂ − b₃x̄₃ − ⋯ − bₚx̄ₚ
Partial regression coefficient bⱼ: "If I increase xⱼ by 1 unit and hold all other variables constant, x₁ changes by bⱼ units." This is why it's called 'partial' — everything else is held fixed.
✏️ Numerical Examples from Notes
Iris Dataset (X₁=Sepal length, X₂=Sepal width, X₃=Petal length, X₄=Petal width):
(a) Linear: X₁.₂₃₄ = 1.856 + 0.65084·X₂ + 0.70913·X₃ − 0.55648·X₄ R² = 0.8586 → 85% variability explained (b) Log-linear: X₁.₂₃₄ = 1.3263 + 2.1734·log X₂ + 1.7634·log X₃ − 0.2210·log X₄ R² = 0.7953 → 79% variability explained → Linear is more efficient (R² 0.8586 > 0.7953)
Another dataset (unnamed in notes):
(a) Linear: x̂₁.₂₃₄ = 0.410581 − 0.012621·x₂ + 0.004287·x₃ + 0.012397·x₄ R² = 0.6911 → 69% variability explained (b) Log: x₁.₂₃₄ = 0.22384 − 0.11186·log x₂ + 0.05320·log x₃ + 0.03831·log x₄ R² = 0.9063 → 90% variability explained → Log regression is more efficient here (R² 0.9063 > 0.6911)
04 7 Important Results on Regression New
📓 From Notes (Recapitulation)
Let X₁.₂₃…ₚ = fitted/predicted value of x₁, and x₁.₂₃…ₚ = x₁ − X₁.₂₃…ₚ = residual.

  1. Mean of fitted values = mean of observed: X̄₁.₂₃…ₚ = x̄₁
  2. Mean residual = 0: x̄₁.₂₃…ₚ = 0
  3. Residuals uncorrelated with predictors: Cov(xᵢ, x₁.₂₃…ₚ) = 0 for i = 2, 3, …, p
  4. Fitted and residual are uncorrelated: Cov(X₁.₂₃…ₚ, x₁.₂₃…ₚ) = 0
  5. Variance of fitted values: V(X₁.₂₃…ₚ) = Cov(x₁, X₁.₂₃…ₚ)
  6. Variance of fitted values (formula):
    V(X₁.₂₃…ₚ) = (1 − |R|/R₁₁) · s₁²
  7. Variance of residuals (formula):
    V(x₁.₂₃…ₚ) = (|R|/R₁₁) · s₁²
💡 Simple Explanation
These 7 results are the multivariate analogues of the well-known OLS regression properties:

ResultWhat it meansWhy it matters
1. Mean(fitted) = mean(observed)Regression passes through the centroid (x̄₁, x̄₂,…,x̄ₚ)The model is unbiased at the mean
2. Mean(residual) = 0Errors average out to zeroNo systematic over/under-prediction
3. Cov(xᵢ, residual) = 0Residuals uncorrelated with all predictorsConfirms OLS optimality
4. Cov(fitted, residual) = 0Signal and noise are orthogonalClassical OLS decomposition
6 & 7. Variance partitionV(x₁) = V(fitted) + V(residual)Total variance split into explained + unexplained
Result 6 & 7 together: V(X₁.₂₃…ₚ) + V(x₁.₂₃…ₚ) = (1−|R|/R₁₁)s₁² + (|R|/R₁₁)s₁² = s₁² = V(x₁). ✓ Variance is perfectly partitioned.
05 Multiple Correlation Coefficient
📓 From Notes
Our purpose is to obtain a measure of the closeness of the relationship between x₁ and x₂, x₃, …, xₚ together.

In bivariate case, the measure of linear correlation is |r|. Similarly, the simple correlation coefficient between x₁ (observed) and X₁.₂₃…ₚ (predicted) serves as a measure of the joint statistical influence of x₂, x₃, …, xₚ on x₁. It is denoted by R₁.₂₃…ₚ and called the multiple correlation coefficient.

Definition:
R₁.₂₃…ₚ = Cov(x₁, X₁.₂₃…ₚ) / √[V(x₁) · V(X₁.₂₃…ₚ)]
Formula:
R₁.₂₃…ₚ = (1 − |R|/R₁₁)^(1/2)
Remarks:
  1. V(X₁.₂₃…ₚ) = R²₁.₂₃…ₚ · s₁²
  2. V(x₁.₂₃…ₚ) = (1 − R²₁.₂₃…ₚ) · s₁²
  3. 0 ≤ R₁.₂₃…ₚ ≤ 1
In the population (theoretical case):
ρ²₁.₂₃…ₚ = 1 − E[V(X₁|X₂,…,Xₚ)] / V(X₁)
💡 Simple Explanation
R₁.₂₃…ₚ is just the ordinary correlation coefficient between the actual x₁ values and the predicted X₁.₂₃…ₚ values. Its square R² tells us what proportion of variance in x₁ is explained by all the other variables together.

Key interpretations:
  • R² = 0 → predictors explain nothing about x₁
  • R² = 1 → perfect prediction, all variance explained
  • R² = 0.85 → "85% of the variation in x₁ is explained by x₂, x₃, …, xₚ"
Note: Unlike simple r which ranges from −1 to +1, R always ranges from 0 to 1 (it measures overall predictability, not direction).
06 Partial Correlation Coefficient & Spurious Correlation
📓 From Notes (Recapitulation — complete theory)
Sometimes the correlation between two variables x₁ and x₂ may be partly or wholly caused by the influence of other variables x₃, x₄, …, xₚ on both x₁ and x₂. We want to know what the correlation between x₁ and x₂ would be if the effects of x₃, x₄, …, xₚ on each of them were eliminated. This is called the partial correlation or net correlation between x₁ and x₂ eliminating x₃, x₄, …, xₚ.

Construction via residuals: Fit the regression of x₁ on x₃, x₄, …, xₚ and x₂ on x₃, x₄, …, xₚ:
X₁.₃₄…ₚ = γ₀ + γ₃x₃ + γ₄x₄ + … + γₚxₚ X₂.₃₄…ₚ = δ₀ + δ₃x₃ + δ₄x₄ + … + δₚxₚ Residuals (parts unaffected by x₃,…,xₚ): x₁.₃₄…ₚ = x₁ − X₁.₃₄…ₚ x₂.₃₄…ₚ = x₂ − X₂.₃₄…ₚ r₁₂.₃₄…ₚ = Corr(x₁.₃₄…ₚ, x₂.₃₄…ₚ)
Formula:
r₁₂.₃₄…ₚ = −R₁₂ / √(R₁₁ · R₂₂)
Remarks:
  1. −1 ≤ r₁₂.₃₄…ₚ ≤ 1
  2. For total r₁₂: x₃,…,xₚ are ignored. For partial r₁₂.₃₄…ₚ: x₃,…,xₚ are taken into account and their effects are sought to be eliminated.
  3. Spurious Correlation: Consider 3 variables x₁, x₂, x₃. If r₁₂.₃ = 0, then:
    (r₁₂ − r₁₃r₂₃) / √[(1−r₁₃²)(1−r₂₃²)] = 0 ⟹ r₁₂ = r₁₃r₂₃ which may ≠ 0
    So x₁ and x₂ have a non-zero correlation when x₃ is ignored, but are uncorrelated when x₃ is accounted for. By ignoring x₃, we introduce a non-zero correlation that was not really there. This is called spurious correlation.
Numerical examples from handwritten notes:
Iris dataset: r₁₂ = −0.1175 → r₁₂.₃₄ = +0.6286 Taking X₃, X₄ into account introduces HIGH POSITIVE correlation Other dataset: r₁₂ = −0.7485 → r₁₂.₃₄ = −0.794 | Cov(x₁,x₂) = −0.748 Taking X₃, X₄ into account introduces SLIGHT additional negative correlation
💡 Simple Explanation
The key idea behind partial correlation:

Imagine you observe that ice cream sales (x₁) and drowning deaths (x₂) are highly positively correlated. Does ice cream cause drowning? No — both are driven by a third variable: hot weather (x₃). Once you control for temperature (eliminate x₃'s effect), the correlation between x₁ and x₂ drops to near zero. That zero is the partial correlation. The original correlation was spurious.

Computation recipe:
1
Regress x₁ on all other variables EXCEPT x₁ and x₂. Get residuals e₁.
2
Regress x₂ on all other variables EXCEPT x₁ and x₂. Get residuals e₂.
3
r₁₂.₃₄…ₚ = ordinary correlation between e₁ and e₂.
4
Or use the formula: −R₁₂/√(R₁₁·R₂₂)
Spurious Correlation: When r₁₂ ≠ 0 but r₁₂.₃ = 0, the observed correlation between x₁ and x₂ is entirely due to their common relationship with x₃. Controlling for x₃ reveals the truth.
Part II — Multivariate Probability Distributions
07 Multivariate Probability Distributions
📓 From Notes (Handwritten)
Let X₁, X₂, …, Xₚ be p random variables.

Case 1 — Discrete: Each Xᵢ takes values xᵢ₁, xᵢ₂, …, xᵢmᵢ (i = 1(1)p).
P{X₁=x₁ⱼ₁, X₂=x₂ⱼ₂, …, Xₚ=xₚⱼₚ} = p_{j₁j₂…jₚ}
The joint pmf f(x₁,…,xₚ) satisfies:
  • f(x₁,…,xₚ) ≥ 0 for all x₁,…,xₚ
  • Σ f(x₁,…,xₚ) = 1
Case 2 — Continuous: Joint pdf f(x₁,…,xₚ) where:
  • f(x₁,…,xₚ) ≥ 0
  • ∫⋯∫ f(x₁,…,xₚ) dx₁…dxₚ = 1
💡 Simple Explanation
Discrete: List all combinations and assign probabilities. Probabilities must be non-negative and sum to 1.
Continuous: Use a density function; probabilities are volumes under a surface. Two rules: non-negative density, total volume = 1.
08 Marginal & Conditional Distributions
📓 From Notes
Marginal pmf of Xᵢ:
g(xᵢ) = Σ f(x₁, x₂,…, xₚ) [sum over all other variables]
Marginal of X₁,…,Xq (q < p):
g(x₁,…,xq) = Σ f(x₁,…,xₚ) [sum over xq₊₁,…,xₚ]
Conditional pmf of X₁ given x₂,…,xₚ:
f(x₁ | x₂,…,xₚ) = f(x₁,…,xₚ) / g(x₂,…,xₚ)
Conditional of X₁,…,Xq given Xq₊₁,…,Xₚ:
f(x₁,…,xq | xq₊₁,…,xₚ) = f(x₁,…,xₚ) / g(xq₊₁,…,xₚ)
Note: In the continuous case, marginal and conditional distributions are obtained similarly (integrate instead of sum).
💡 Simple Explanation
Marginal: "Sum out" or "integrate out" the variables you don't want to study. Like looking at row/column totals in a table.

Conditional: Distribution of some variables given others are fixed. Formula = Joint ÷ Marginal of the given variables.
09 Mean Vector & Dispersion Matrix
📓 From Notes (Handwritten)
E(Xᵢ) = μᵢ, V(Xᵢ) = σᵢ², Cov(Xᵢ, Xⱼ) = ρᵢⱼ σᵢ σⱼ, |ρᵢⱼ| ≤ 1

Mean Vector:
μ = E(X) = (μ₁, μ₂, …, μₚ)'
Dispersion (Variance-Covariance) Matrix:
Σ = E[(X−μ)(X−μ)'] ⎡ V(X₁) Cov(X₁,X₂) … Cov(X₁,Xₚ) ⎤ = ⎢ Cov(X₂,X₁) V(X₂) … Cov(X₂,Xₚ) ⎥ ⎣ Cov(Xₚ,X₁) Cov(Xₚ,X₂) … V(Xₚ) ⎦ ⎡ σ₁² σ₁₂ σ₁₃ … σ₁ₚ ⎤ = ⎢ σ₂₁ σ₂² σ₂₃ … σ₂ₚ ⎥ where σᵢⱼ = Cov(Xᵢ,Xⱼ) ⎣ σₚ₁ σₚ₂ σₚ₃ … σₚ²⎦
Notes:
  1. Σ is positive definite.
  2. Σ is also called the Variance-Covariance Matrix.
  3. Can equivalently be represented by the Correlation Matrix R (with 1s on diagonal, ρᵢⱼ off-diagonal).
  4. Both Σ and R are symmetric.
💡 Simple Explanation
Mean vector: one mean per variable, stacked in a column.

Dispersion matrix Σ: a p×p table where diagonal = variances, off-diagonal = covariances. Symmetric (because Cov(Xᵢ,Xⱼ) = Cov(Xⱼ,Xᵢ)) and positive definite (variances can't be negative).

Correlation matrix R: standardised Σ — divide each covariance by the product of the two SDs. Diagonal becomes 1, off-diagonal becomes ρᵢⱼ ∈ [−1, 1].
10 Important Results on Mean Vector & Dispersion Matrix
📓 From Notes
Let X be a random vector with mean vector μ and dispersion matrix Σ.

Result 1 — For any real vector a:
E(a'X) = a'μ V(a'X) = a'Σa
Result 2 — For any real matrix C:
E(CX) = Cμ D(CX) = CΣC'
Joint CDF:
F(x₁,x₂,…,xₚ) = P(X₁ ≤ x₁, X₂ ≤ x₂, …, Xₚ ≤ xₚ)
💡 Simple Explanation
Result 1: Any linear combination a'X = a₁X₁+a₂X₂+⋯ → mean = a'μ, variance = a'Σa (quadratic form).
Result 2: Matrix transform CX → mean = Cμ, dispersion = CΣC'. Used constantly to derive distributions of transformed variables.
11 Multinomial Distribution
📓 From Notes
Let X₁, X₂,…,Xk be jointly distributed discrete random variables. They follow a multinomial distribution if:
f(x₁,x₂,…,xk) = n! / (x₁! x₂! ⋯ xk! (n−Σxᵢ)!) · p₁^x₁ p₂^x₂ ⋯ pk^xk · (1−Σpᵢ)^(n−Σxᵢ) where xᵢ = 0,1,…,n ∀i, Σxᵢ ≤ n, 0 < pᵢ < 1, Σpᵢ < 1
Alternative (k+1 variable) form with Σpᵢ = 1 and Σxᵢ = n — avoid this form as the dispersion matrix becomes singular.

Special cases:
Trinomial: f(x,y,z) = n!/(x!y!z!) · p^x q^y r^z, x+y+z=n, p+q+r=1 Binomial (k=1): f(x) = C(n,x)·p^x·q^(n-x)
💡 Simple Explanation
Multinomial = Binomial extended to k > 2 outcomes. Roll a die n times instead of flipping a coin.
Negative covariance: Cov(Xᵢ, Xⱼ) = −npᵢpⱼ < 0. Always negative — more of one category necessarily means less of another.
12 Properties of Multinomial Distribution ★ Exam Important
📓 From Notes (Handwritten)
Let (X₁, X₂,…,Xk) ~ Multinomial(n, p₁, p₂,…,pk).

P1: Marginal of Xᵢ ~ Binomial(n, pᵢ)
P2: Marginal of (X₁,…,Xq) for q < k ~ Multinomial(n, p₁,…,pq)
P3: Conditional of X₁ | X₂=x₂,…,Xk=xk ~ Binomial(n−Σᵢ₌₂ᵏ xᵢ, p₁*) where p₁* = p₁/(1−Σᵢ₌₂ᵏ pᵢ)
P4: Conditional of (X₁,…,Xq) | Xq₊₁=xq₊₁,…,Xk=xk ~ Multinomial(n−Σᵢ₌q₊₁ᵏ xᵢ, p₁*,…,pq*) where pⱼ* = pⱼ/(1−Σᵢ₌q₊₁ᵏ pᵢ)
P5: S = X₁+X₂+⋯+Xq (q ≤ k) ~ Binomial(n, Σᵢ₌₁q pᵢ)
P6: Cov(Xᵢ, Xⱼ) = −npᵢpⱼ for i ≠ j
P7 ★: True regression of X₁ on X₂,…,Xk is LINEAR:
E(X₁|x₂,…,xk) = (n−x₂−x₃−⋯−xk)·p₁* [linear in x₂,…,xk]
Multiple Correlation for Multinomial:
ρ²₁.₂₃…k = (p₁*−p₁)/(1−p₁) ∴ ρ₁.₂₃…k = √[(p₁*−p₁)/(1−p₁)]
Partial Correlation for Multinomial:
ρ₁₂.₃₄…k = −√(p₁p₂) / [√(1−p₁−p₃−⋯−pk) · √(1−p₂−p₃−⋯−pk)] = −np₁p₂* / √[np₁(1−p₁*)·np₂(1−p₂*)]
💡 Simple Explanation
PropertyPlain English
P1 (Marginal = Binomial)Each individual count is binomial — one category either happens or not
P2 (Subset is Multinomial)Any group of categories from a multinomial is itself multinomial
P3 (Conditional = Binomial)Fix all other counts → remaining count is binomial with adjusted probability p₁*
P5 (Sum = Binomial)Combine categories → binomial with combined probability
P6 (Negative Covariance)More of one → less of another (they compete for the fixed total n)
P7 ★ (Linear Regression)The conditional expectation E(X₁|others) is linear in the other xᵢ values
p₁* interpretation: p₁* = p₁/(1−Σpᵢ) is the conditional probability that a trial falls in category 1, given it didn't fall in categories 2,…,k. It's p₁ rescaled to the remaining probability space.
Part III — Multivariate Normal & Sampling
13 Multivariate Normal Distribution (MVN)
📓 From Notes (Typed Notes — MVN)
Univariate Normal:
f(x) = 1/(σ√(2π)) · exp[−(x−μ)²/(2σ²)], −∞ < x < ∞
p-variate Normal Density:
f(x) = 1/((2π)^(p/2)|Σ|^(1/2)) · exp[−½(x−μ)'Σ⁻¹(x−μ)] −∞ < xᵢ < ∞ for i = 1,…,p Notation: X ~ Nₚ(μ, Σ)
In the univariate case: scalar x, mean μ, variance σ². In the multivariate case: x is a vector, μ is a mean vector, σ² is replaced by dispersion matrix Σ (p×p symmetric positive definite). The squared term (x−μ)²/σ² becomes the quadratic form (x−μ)'Σ⁻¹(x−μ). Bivariate Normal (BVN), p = 2:
f(x₁,x₂) = 1/(2π√σ₁₁ √σ₂₂ √(1−ρ₁₂²)) · exp{−1/(2(1−ρ₁₂²))[(x₁−μ₁)²/σ₁₁ + (x₂−μ₂)²/σ₂₂ − 2ρ₁₂(x₁−μ₁)/√σ₁₁ · (x₂−μ₂)/√σ₂₂]}
Special Property of BVN:
If ρ₁₂ = 0: f(x₁,x₂) = f_{X₁}(x₁) · f_{X₂}(x₂) ∴ X₁ and X₂ are INDEPENDENT In BVN (and MVN): Uncorrelated ⟺ Independent
Note: Under multivariate case, probabilities are represented by volumes under the surface over regions defined by intervals of xᵢ values.

14 Results on MVN: Linear Combinations & Partitioning
📓 From Notes (Typed Notes — MVN Results)
Result 1 — Linear Combinations: Let X ~ Nₚ(μ, Σ) and A be any m×p matrix of rank m ≤ p. Then:
Y = AX ~ Nₘ(Aμ, AΣA')
Remark: If m=1, the scalar a'X has univariate Normal with mean a'μ and variance a'Σa. The joint distribution of any subset of elements of X is multivariate normal. Caution: Normal marginals do NOT necessarily imply joint normality.

Result 2 — Partitioning: Let X ~ Nₚ(μ, Σ) with partition:
X = (X⁽¹⁾ : q×1, X⁽²⁾ : (p−q)×1) μ = (μ⁽¹⁾, μ⁽²⁾) Σ = | Σ₁₁ Σ₁₂ | (q×q and (p−q)×(p−q) blocks) | Σ₂₁ Σ₂₂ | Independence: X⁽¹⁾ ⊥ X⁽²⁾ ⟺ Σ₁₂ = 0 Conditional distributions: X⁽²⁾ | X⁽¹⁾=x⁽¹⁾ ~ N_{p−q}(μ⁽²⁾ + Σ₂₁Σ₁₁⁻¹(x⁽¹⁾−μ⁽¹⁾), Σ₂₂.₁) where Σ₂₂.₁ = Σ₂₂ − Σ₂₁Σ₁₁⁻¹Σ₁₂ X⁽¹⁾ | X⁽²⁾=x⁽²⁾ ~ N_q(μ⁽¹⁾ + Σ₁₂Σ₂₂⁻¹(x⁽²⁾−μ⁽²⁾), Σ₁₁.₂) where Σ₁₁.₂ = Σ₁₁ − Σ₁₂Σ₂₂⁻¹Σ₂₁
💡 Simple Explanation
Result 1: MVN is closed under linear transformations. Any linear combination of normal variables is normal. This makes the MVN incredibly easy to work with.

Result 2 (Partitioning): Split variables into two groups. Each group, conditioned on the other, is still MVN with:
  • Conditional mean = original mean + regression correction: μ⁽²⁾ + Σ₂₁Σ₁₁⁻¹(x⁽¹⁾−μ⁽¹⁾)
  • Conditional variance = Σ₂₂ − Σ₂₁Σ₁₁⁻¹Σ₁₂ (always smaller — knowing more reduces uncertainty)
Independence rule in MVN: Two subvectors of MVN are independent if and only if their cross-covariance block Σ₁₂ = 0 (null matrix).
15 Sampling from MVN — MLE & Wishart Distribution
📓 From Notes
MVN is completely specified by μ and Σ. Sample of n observations x₁,…,xₙ from Nₚ(μ,Σ), where xₐ = (xₐ₁,…,xₐₚ)'. (First subscript = unit, second = variable.)

Sample Mean Vector:
x̄ = (1/n)Σₐ xₐ = (x̄₁, x̄₂, …, x̄ₚ)'
SSCP Matrix (Sum of Squares and Cross Products):
A = Σₐ (xₐ−x̄)(xₐ−x̄)' = [Σₐ(xₐᵢ−x̄ᵢ)(xₐⱼ−x̄ⱼ)]
MLE:
μ̂ = x̄ (unbiased) Σ̂ = A/n (biased)
Theorem: x̄ ~ Nₚ(μ, Σ/n), independent of Σ̂.

Wishart Distribution: If M = X'X where X_{n×p} is from Nₚ(0,Σ), then M ~ Wₚ(Σ, m).
If Σ=I: standard form. If p=1, Σ=1: M = Σxᵢ² ~ χ²_m
Properties:
  1. Sum of independent Wisharts (same Σ, d.f. m₁,…,mk): ~ Wₚ(Σ, Σmᵢ)
  2. E(M) = mΣ
  3. M ~ Wₚ(Σ,m), B a p×q matrix → B'MB ~ Wq(B'ΣB, m)
  4. A = nΣ̂ ~ Wₚ(Σ, n−1)
💡 Simple Explanation
Wishart = multivariate χ². Just as Σxᵢ² ~ χ² in the univariate case, the SSCP matrix A follows a Wishart distribution.
Key analogy: Univariate: (n−1)s² ~ σ²·χ²_{n-1}. Multivariate: A = nΣ̂ ~ Wₚ(Σ, n−1). So the unbiased estimator is A/(n−1).

📓 From Notes
Z = m · d' M⁻¹ d where d ~ Nₚ(0, Iₚ) and M ~ Wₚ(I, m) (independent) → Z ~ T²(p, m) [Hotelling's T²]
Relationship with F:
T²(p, m) = [mp/(m−p+1)] · F_{p, m−p+1} i.e. [(m−p+1)/(mp)] · Z ~ F_{p, m−p+1}

Reading the plot:

  • Tall block → that subcategory is more common within the column

18 Scatterplot Matrix (SPLOM / Pair Plot) New Viz
📓 From Notes (Note_19_01_26)
A Scatterplot Matrix (SPLOM or Pair Plot) is a powerful visualization showing pairwise relationships between multiple variables in a dataset through a grid of individual scatterplots. It reveals correlations, patterns, and outliers across all variable combinations. Diagonal plots often show variable distributions.

Uses in Data Analysis:
  • Correlation Detection: Quickly spot strong positive, negative, or no correlations between variables.
  • Pattern Identification: Uncover non-linear relationships or clusters.
  • Outlier Detection: Identify unusual data points in multiple dimensions.
  • Feature Selection: Inform regression or modelling by identifying potentially multicollinear predictors.
💡 Simple Explanation
Imagine a p×p grid of scatterplots. The cell at row i, column j shows the scatterplot of variable xᵢ vs xⱼ. The diagonal (row i = col i) shows the distribution of xᵢ itself (histogram or density).

What to look for:
  • Tight upward band → strong positive correlation
  • Tight downward band → strong negative correlation
  • Circular cloud → near-zero correlation
  • Curved pattern → non-linear relationship
  • Isolated point far from cloud → outlier
Practical tip: With p variables, you get p(p−1)/2 unique scatterplots. With p=4 (Iris), that's 6 plots shown simultaneously. With p=10, that's 45 plots — so SPLOM scales well for initial exploration.
19 Bivariate Q-Q Plots New Viz
📓 From Notes (Note_19_01_26)
A Q-Q (quantile–quantile) plot compares the quantiles of two distributions. In the bivariate case, instead of comparing one variable to a theoretical distribution, you compare two sets of paired data. It helps assess whether two variables follow the same distributional shape or whether one systematically differs.

Applications of Q-Q Plots:
  • Assessing Distributional Assumptions: Visually inspect whether a dataset follows a specific distribution (e.g., normal). Crucial for statistical inference.
  • Detecting Outliers: Points that deviate significantly from the diagonal line indicate outliers.
  • Comparing Distributions: Are two datasets from the same distribution? Points on the diagonal → yes.
  • Assessing Normality: Points on the y=x line → normal. Systematic curve → skewed or heavy-tailed.
  • Model Validation: Check if model residuals are normally distributed (econometrics, ML).
  • Quality Control: Monitor distribution of measured values over time or across batches.
💡 Simple Explanation
A Q-Q plot sorts both datasets and plots the i-th quantile of one against the i-th quantile of the other.

Reading the plot:
  • Points follow the diagonal line y=x → same distribution
  • Points curve upward at the ends → heavier tails than reference (leptokurtic)
  • Points curve inward → lighter tails (platykurtic)
  • S-shape → one distribution is skewed relative to the other
  • Points jump off the line at one end → outliers in the tail
Most common use: Normal Q-Q plot — plot your data's quantiles against normal distribution quantiles. If the data is normal, all points fall on a straight line.
20 Spider / Radar Charts New Viz
📓 From Notes (Note_19_01_26)
A spider chart (also called a radar chart) is used when we want to display data across several unique dimensions simultaneously. Dimensions are usually quantitative and typically normalized to the same range. The coordinate system used is polar coordinates (not Cartesian).

Each dimension gets its own axis radiating from the center. The data point for each dimension is marked on its axis, and all points are connected to form a polygon. The area of the polygon represents the overall "score."

Advantages:
  • Allows comparison of multiple dimensions in a compact space.
  • Visually engaging and easy to present to audiences.
  • Good for comparing profiles of two or more entities.

Example (Students): For student Arthur, dimensions might be: Final exam grade, Midterm grade, Assignments completed, Hours studied, Days attended. Each dimension gets a spoke, and Arthur's values are plotted to form his "performance polygon."
💡 Simple Explanation
Imagine a clock face. Each hour mark represents one variable. Draw a dot on each spoke based on the variable's value (closer to centre = smaller value). Connect the dots — that's the spider chart.

When to use:
  • Comparing performance profiles (e.g., two athletes across 5 skills)
  • Showing strengths and weaknesses across multiple dimensions
  • When all variables are on comparable scales (or are normalized)
When NOT to use:
  • Many variables (>8) — becomes hard to read
  • When axis order matters (spider charts are sensitive to dimension ordering)
  • For precise numerical comparison (bar charts are better)
21 DD Plots (Depth versus Depth) New Viz
📓 From Notes (Note_19_01_26)
The DD (Depth vs Depth) plot is a graphical tool for comparing two multivariate datasets regardless of dimension. It transforms high-dimensional data into a simple 2D scatter plot.

Key Concept — Data Depth: Measures how "central" or "outlying" a data point is within a multivariate sample. Various depth functions exist: halfspace depth, simplicial depth.

Construction:
1
Compute the depth of every point (from both samples) with respect to distribution F.
2
Compute the depth of every point with respect to distribution G.
3
Plot pairs (Depth(z, F), Depth(z, G)) for all points z in the combined sample.
Interpretation:
  • Identical distributions (F=G): Points cluster tightly around the identity line y=x.
  • Location shift only: Points form a "leaf" shape deviating from the diagonal. Direction shows which distribution has larger location parameter.
  • Scale or skewness differences: Different patterns of deviation from the diagonal.
Applications:
  • Graphical comparison: Nonparametric visual comparison of two multivariate samples without distributional assumptions.
  • Hypothesis testing: Visual patterns can be formalized into permutation tests for homogeneity, location, or scale differences.
  • Outlier detection: Compare sample to a theoretical distribution to identify multivariate outliers.
  • Classification (DD-classifier): Find an optimal curve in 2D depth space to separate classes.
💡 Simple Explanation
The problem with comparing two high-dimensional datasets is you can't simply plot them. The DD plot solves this by reducing everything to a single number per point: its depth (how central the point is in each dataset).

Depth intuition: A point that is right in the middle of a cloud of data has high depth. A point at the edge has low depth.

Reading the DD plot:
  • Points on the diagonal → both datasets treat this point as equally central → same distribution
  • Points above diagonal → point is more central in G than F
  • Points below diagonal → point is more central in F than G
  • Leaf shape → location shift (one group's centre is shifted)
Key advantage: Completely nonparametric (no normality assumption needed) and works in any number of dimensions.
22 Parallel Coordinate Plots New Viz
📓 From Notes (Note_19_01_26)
A parallel coordinate plot is a data visualization technique used to plot and analyse multivariate data. Each data point is represented as a single continuous line traversing across several parallel, vertical axes, with each axis representing a different variable.

Key Components:
  • Parallel Axes: Equally spaced vertical lines, each corresponding to a specific variable. Each axis has its own scale or is normalized.
  • Data Lines (Polylines): Each line = one observation. Where a line intersects an axis = value of that variable for that observation.
Interpretation:
  • Positive Correlation: Lines between two adjacent axes appear parallel (same direction).
  • Negative Correlation: Lines between two adjacent axes cross frequently, forming an "X" shape.
  • No Correlation: Lines cross randomly or show no pattern.
  • Clusters: Similar data points form distinct bands or groups of parallel lines.
  • Outliers: Lines that conspicuously diverge from the main flow.
Applications: Data analysis, engineering, scientific research for multidimensional data exploration, feature analysis, and anomaly detection. Supported in Plotly, Pandas, ELKI.
💡 Simple Explanation
Instead of plotting variables on x-y axes (which only works for 2 variables), parallel coordinate plots put each variable on its own vertical axis and draw a line connecting each observation's values across all axes.

Think of it as: Each person in a dataset is a coloured thread woven through vertical poles (variables). If most threads go in the same direction between two poles → positive correlation. If they cross → negative correlation.

Example with 4 variables (Iris): Draw 4 vertical axes (Sepal length, Sepal width, Petal length, Petal width). Each flower is one line connecting its 4 measurements. Species form distinct bands of lines, showing natural clusters.
Limitation: Only adjacent axes are visually compared easily. Reordering axes can reveal different patterns — try different orderings to explore the data.
23 Trellis Displays New Viz
📓 From Notes (Note_19_01_26)
Trellis charts (also known as multiple mini-charts or faceted plots) are a powerful data visualization technique that allows for comparison of multiple subsets of data within a single visualization.

How it works: The data is divided into multiple small charts ("trellises"), each representing a specific subcategory. Each trellis can be further divided into smaller subplots using traditional charting methods (bar charts, line charts, scatter plots).

Main advantages:
  • Comparison of many subsets of data in a single visualization.
  • Useful when data has many dimensions.
  • Helps identify patterns and trends difficult to see in a single large chart.
  • Supports bar charts, line charts, and scatter plots as sub-plots.
💡 Simple Explanation
A trellis display is like making the same chart multiple times — once for each subgroup of your data — and displaying them side by side on a grid (the "trellis").

Example: You have sales data for 4 regions × 3 years × 2 product types. Instead of one messy chart, you make a 4×3 grid of charts, each showing one region-year combination, with both product types in each chart.

Key principle: Each mini-chart has the same scale and format, so you can make fair visual comparisons across all the panels at a glance.
Modern name: In ggplot2 (R) this is called "faceting" (facet_wrap or facet_grid). In Plotly it's "subplots". The trellis display principle is the same.

24 Problem Set 1 — Multivariate Descriptive Statistics New
📓 Problem Statement (Multivariate_1.docx — Date: 01.02.22)
Per capita demand for chicken (x₁, kg), price of chicken (x₂, Rs/kg), price of fish (x₃, Rs/kg), per capita income (x₄, thousand Rs at 1993-94 prices) across different Indian states.

Tasks:
  1. Fit a multiple linear regression of x₁ on x₂, x₃, x₄. Obtain a measure of efficacy (R²).
  2. Fit a multiple linear regression of x₁ on ln x₂, ln x₃, ln x₄. Obtain R².
  3. Compare regressions (a) and (b) in terms of their efficacy.
  4. Compute the total (simple) correlation coefficient between x₁ and x₂.
  5. Compute the partial correlation coefficient between x₁ and x₂ eliminating x₃ and x₄.
  6. Comment on (d) and (e).
✏️ How to Solve — Method Guide
(a) & (b) Fitting regression + R²:
1
Compute means x̄ᵢ and standard deviations sᵢ for each variable (use log-transformed x₂, x₃, x₄ for part b).
2
Compute the 4×4 correlation matrix R with all pairwise rᵢⱼ = sᵢⱼ/(sᵢsⱼ).
3
Compute cofactors R₁₁, R₁₂, R₁₃, R₁₄ of R.
4
Apply: bⱼ = −(R₁ⱼ/R₁₁)·(s₁/sⱼ) for j=2,3,4. Then b₀ = x̄₁ − b₂x̄₂ − b₃x̄₃ − b₄x̄₄.
5
R² = 1 − |R|/R₁₁. This is the efficacy measure.

(c) Comparison: Higher R² = more efficient equation. If R²(b) > R²(a), log-linear is better.

(d) Total correlation r₁₂:
r₁₂ = s₁₂/(s₁s₂) [direct formula]
(e) Partial correlation r₁₂.₃₄:
r₁₂.₃₄ = −R₁₂/√(R₁₁·R₂₂) where R₁₂ = cofactor of r₁₂ in R, R₁₁ = cofactor of r₁₁, R₂₂ = cofactor of r₂₂
(f) Comment: Compare r₁₂ (ignores x₃, x₄) with r₁₂.₃₄ (eliminates x₃, x₄). If very different, x₃ and/or x₄ are confounders. If r₁₂ ≠ 0 but r₁₂.₃₄ ≈ 0, the simple correlation is spurious.
Context: We'd expect the demand for chicken (x₁) to be negatively related to chicken price (x₂) — but this might be confounded by income (x₄) and fish price (x₃). Partial correlation isolates the pure price effect.
✏️ Related Numerical Results from Notes (similar dataset)
Linear: R² = 0.6911 → 69% explained Log-linear: R² = 0.9063 → 90% explained (more efficient) r₁₂ = −0.748 (simple correlation: negative, as expected) r₁₂.₃₄ = −0.794 (partial correlation: slightly more negative after controlling x₃, x₄) → x₃ and x₄ introduce slight masking of the true negative relationship
25 Problem Set 2 — Multinomial Distribution New
📓 Problem Statements (Multinomial_Distribution.docx)
Q1. 10 distinguishable balls distributed at random over 4 cells (each cell can receive any number). Xᵢ = number of balls in cell i (i=1,2,3,4). Find:
  • (i) P{X₁=1, X₂=2, X₃=3, X₄=4}
  • (ii) Expectation and variance of balls in cell 1 when X₂=2, X₃=3
  • (iii) Partial correlation coefficient between X₁ and X₂ eliminating X₃, X₄
Q2. A die with probabilities p₁=0.10, p₂=0.30, p₃=0.25, p₄=0.20, p₅=0.09, p₆=0.06 is thrown 20 times. Let fᵢ = number of occurrences of face i. Find:
  • Correlation coefficient between f₂ and f₅, and partial correlation between f₂ and f₅ eliminating f₁ and f₃.
  • Conditional variance of f₂ when f₁=3, f₃=3, f₄=3, f₅=3 (i.e., fᵢ=3 for i=1,3,4,5).
Q3. A document has 10 words, each classified as positive (p=0.4), neutral (p=0.3), or negative (p=0.3). Find:
  • P(4 positive, 3 neutral, 3 negative)
  • Expected number and variance of positive words
  • Covariance between positive and neutral word counts
✏️ Complete Solutions
Q1 Setup: Each ball goes into any of 4 cells equally likely → p₁=p₂=p₃=p₄=1/4. n=10. (X₁,X₂,X₃,X₄) ~ Multinomial(10, 1/4, 1/4, 1/4, 1/4). Q1(i):
P{X₁=1,X₂=2,X₃=3,X₄=4} = 10! / (1!·2!·3!·4!) · (1/4)¹·(1/4)²·(1/4)³·(1/4)⁴ = 3628800 / (1·2·6·24) · (1/4)^10 = 3628800 / 288 · 1/1048576 = 12600 / 1048576 ≈ 0.01202
Q1(ii): Given X₂=2, X₃=3 (so x₂+x₃=5 balls assigned to cells 2,3). Remaining n−x₂−x₃ = 10−2−3 = 5 balls go into cells 1 and 4.
By Property 3: X₁|X₂=2,X₃=3 ~ Binomial(5, p₁*) where p₁* = p₁/(1−p₂−p₃) = (1/4)/(1−1/4−1/4) = (1/4)/(1/2) = 1/2 E(X₁|X₂=2,X₃=3) = 5 · (1/2) = 2.5 V(X₁|X₂=2,X₃=3) = 5 · (1/2) · (1/2) = 1.25
Q1(iii): Partial correlation r₁₂.₃₄ using Multinomial formula:
ρ₁₂.₃₄…k = −√(p₁p₂) / [√(1−p₁−p₃−p₄) · √(1−p₂−p₃−p₄)] = −√(1/4 · 1/4) / [√(1−1/4−1/4−1/4) · √(1−1/4−1/4−1/4)] = −(1/4) / [√(1/4) · √(1/4)] = −(1/4) / (1/4) = −1
This means X₁ and X₂ are perfectly negatively correlated after removing X₃ and X₄ effects. (Makes sense — if 3 cells are fixed, the 4th is completely determined.)
Q2 Setup: (f₁,f₂,f₃,f₄,f₅,f₆) ~ Multinomial(20; 0.10,0.30,0.25,0.20,0.09,0.06). n=20. Correlation between f₂ and f₅:
Cov(f₂,f₅) = −n·p₂·p₅ = −20·0.30·0.09 = −0.54 V(f₂) = n·p₂·(1−p₂) = 20·0.30·0.70 = 4.2 V(f₅) = n·p₅·(1−p₅) = 20·0.09·0.91 = 1.638 r₂₅ = −0.54 / √(4.2·1.638) = −0.54 / √6.8796 = −0.54 / 2.6230 ≈ −0.206
Partial correlation r₂₅ eliminating f₁ and f₃:
ρ₂₅.₁₃₄₆ = −√(p₂p₅) / [√(1−p₂−p₁−p₃) · √(1−p₅−p₁−p₃)] = −√(0.30·0.09) / [√(1−0.30−0.10−0.25) · √(1−0.09−0.10−0.25)] = −√(0.027) / [√(0.35) · √(0.56)] = −0.1643 / [0.5916 · 0.7483] = −0.1643 / 0.4427 ≈ −0.371
Conditional variance of f₂ given f₁=3, f₃=3, f₄=3, f₅=3:
Fixed: f₁+f₃+f₄+f₅ = 3+3+3+3 = 12 balls used. Remaining: n − 12 = 20 − 12 = 8 balls for f₂ and f₆. p₂* = p₂/(1−p₁−p₃−p₄−p₅) = 0.30/(1−0.10−0.25−0.20−0.09) = 0.30/0.36 ≈ 0.8333 f₂|given ~ Binomial(8, 0.8333) V(f₂|f₁=3,f₃=3,f₄=3,f₅=3) = 8 · 0.8333 · (1−0.8333) = 8 · 0.8333 · 0.1667 ≈ 1.111
Q3 Setup: n=10 words, p₁=0.4 (positive), p₂=0.3 (neutral), p₃=0.3 (negative). Let X=positive, Y=neutral, Z=negative. (X,Y,Z) ~ Trinomial(10; 0.4, 0.3, 0.3). (i) P(X=4, Y=3, Z=3):
